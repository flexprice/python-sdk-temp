# InvoiceType

invoice_type filters by the nature of the invoice (SUBSCRIPTION, ONE_OFF, or CREDIT)
Use this to separate recurring charges from one-time fees or credit adjustments


## Values

| Name           | Value          |
| -------------- | -------------- |
| `SUBSCRIPTION` | SUBSCRIPTION   |
| `ONE_OFF`      | ONE_OFF        |
| `CREDIT`       | CREDIT         |
